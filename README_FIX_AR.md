# نسخة جاهزة بعد الإصلاح

## ما الذي تم إصلاحه
- الصفحة الرئيسية أصبحت هي `index.html`
- تم فصل لوحة التحكم عن الصفحة الرئيسية
- تم حذف أزرار/تبويبات المعاينة من الواجهة العامة
- تمت إضافة حماية بسيطة بكلمة مرور لمسار `/dashboard`

## بيانات الدخول الافتراضية للوحة التحكم
- المستخدم: `admin`
- كلمة المرور: `123456`

## الأفضل قبل النشر
غيّر بيانات الدخول من إعدادات Vercel:
- `DASHBOARD_USER`
- `DASHBOARD_PASS`

## الرفع إلى GitHub
ارفع محتويات هذا المجلد إلى جذر المستودع ثم أعد النشر في Vercel.
